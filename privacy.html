隐私政策

最后更新：2026 年 7 月 12 日 · 适用于 PageLens v1.1.0

核心承诺：PageLens 不收集、不存储、不出售任何用户个人数据。页面内容仅在用户主动触发摘要时临时处理，API Key 本地加密保存。所有数据处理均在用户设备本地完成（内置 AI 模式），或发送至用户自行选择和配置的 LLM API 服务。

  • 新增 Chrome 内置 AI 说明：数据完全在本地处理，不离开设备

1. 概述

PageLens — AI 智能网页摘要（以下简称"本扩展"）是一款 Chrome 浏览器扩展，为用户当前浏览的网页生成 AI 结构化摘要。我们高度重视用户隐私，本政策详细说明本扩展如何处理数据。

2. 数据收集

本扩展不收集以下任何类型的数据：

数据类型是否收集
个人身份信息（姓名、邮箱、电话）否
银行卡或支付信息否
信用信息否
用户地理位置否
网页浏览历史记录否
用户行为追踪数据否
Cookie 或追踪标识符否

3. 数据使用

本扩展涉及以下数据处理行为：

3.1 页面内容提取

当用户主动点击扩展图标或点击"刷新"/"一键摘要"按钮时，本扩展通过 chrome.scripting.executeScript 在当前页面上下文中注入提取函数，读取页面的可见文本内容。提取的内容：

  • 仅在用户主动触发时执行，不会自动运行
  • 提取后根据用户选择的引擎发送至对应 AI 服务，不在本地持久化存储
  • 摘要生成完成后，页面原文内容不会被保留

3.2 AI 引擎数据处理

本扩展支持三种 AI 引擎，数据处理方式各不相同：

3.2.1 Chrome 内置 AI（推荐）

隐私最优选：使用 Chrome 内置的 Gemini Nano 模型在本地运行 AI 推理，所有数据处理完全在用户设备上完成，不发送任何网络请求。

  • 通过 chrome.languageModel API（Chrome 138+ 扩展稳定 API）调用
  • 页面内容仅在本地 GPU/CPU 上推理，不离开用户设备
  • AI 生成的摘要直接返回到扩展侧边栏
  • 仅调用 availability() 检测可用性，不传输任何数据
  • 不收集任何用户数据或模型使用信息

3.2.2 Gemini 云端免费额度

提取的页面文本将通过 HTTPS 加密连接发送至 Cloudflare Worker 代理，由代理转发至 Google Gemini API。代理服务的处理方式：

  • 页面文本内容：直接转发至 Gemini API，代理不存储、不记录、不分析
  • AI 生成的摘要：直接流式转发回客户端，代理不保留
  • 用户 IP 地址：仅用于限流计数（rate:{IP}:{date} → 计数），24 小时后自动删除
  • API Key：存储在 Cloudflare Worker Secret 中，不暴露给客户端
  • 代理不记录请求内容或响应

3.2.3 自定义 API Key

提取的页面文本将通过 HTTPS 加密连接发送至用户自行选择和配置的 LLM API 端点。本扩展支持的 LLM 提供商包括：

  • OpenAI (api.openai.com)
  • Anthropic Claude (api.anthropic.com)
  • Google Gemini (generativelanguage.googleapis.com)
  • DeepSeek (api.deepseek.com)

用户也可配置自定义 API 端点。API 请求直接从用户浏览器发送至对应服务商，数据的使用受各 LLM 服务商自身隐私政策约束。

3.3 API Key 存储

用户输入的 LLM API Key 使用 AES-GCM 算法加密后存储在 chrome.storage.local 中。加密密钥由用户设备生成，不以明文形式存储或传输。API Key 仅在发起 LLM API 请求时解密使用，不会发送至任何第三方服务器。

3.4 本地配额追踪

免费模式下，本扩展在本地记录每日使用次数，用于显示配额进度和避免超额请求：

  • 存储位置：chrome.storage.local，键 freeTierQuota
  • 存储内容：每日已用次数、限额、日期
  • 数据仅存储在用户浏览器本地，不同步至云端，不传输到任何服务器
  • 跨天后自动重置

3.5 用户偏好设置

以下设置信息存储在本地 chrome.storage.local 中，不会同步至云端：

  • 选择的 LLM 提供商和接入模式
  • 免费引擎偏好（自动/内置AI/Gemini云端）
  • 摘要语言偏好
  • 输出格式偏好（Markdown/纯文本/HTML）
  • 功能开关状态（流式输出、Markdown 渲染等）
  • 自定义 Prompt 模板

6. 第三方服务

本扩展可能与以下第三方服务进行通信：

  • Chrome 内置 AI（本地运行，无网络通信）
  • Cloudflare Worker 代理（免费模式下中转请求至 Gemini API，代理不存储数据）
  • OpenAI 隐私政策
  • Anthropic 隐私政策
  • Google 隐私政策
  • DeepSeek 隐私政策

7. 数据安全

本扩展采取以下措施保护用户数据：

  • API Key 使用 AES-GCM 算法加密存储
  • 所有 LLM API 请求通过 HTTPS 加密传输
  • 内置 AI 模式下数据完全不离设备
  • 代理服务不存储、不记录、不分析任何请求内容
  • 页面内容提取后不在本地持久化存储
  • 不使用任何第三方追踪或分析服务
  • 配额数据仅存储在浏览器本地

8. 儿童隐私

本扩展不面向 13 岁以下儿童，也不会 knowingly 收集儿童的个人信息。

9. 数据控制权

用户对自身数据拥有完全控制权：

  • 用户可在设置中自由选择 AI 引擎（内置 AI / 免费云端 / 自定义 Key）
  • 使用内置 AI 时，数据完全不离开用户设备
  • 用户可随时在扩展设置中删除 API Key
  • 卸载扩展将自动清除所有本地存储的数据
  • 用户可选择不配置任何 API Key，此时仅使用免费模式或内置 AI

10. 政策更新

如本隐私政策发生变更，我们将在此页面上更新"最后更新"日期。重大变更将通过扩展更新说明通知用户。
